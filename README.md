2. Arima 정리

-> prophet의 성능이 크게 좋지 않아서 추세만 확인하기 위해 arima모델을 사용함.    
-> 모델 자체의 구현이 간단한만큼 다양한 경우의 수를 생각하여 31개의 모델을 활용함.  
-> (0,0,0)~(5,1,5) 사이의 수를 활용하여 12개의 set에 모두 사용할수 있는 set은 12개.
